palm-webos-development-tmbundle by devilx

Palm webOS (formerly Palm Pre) Development Bundle for TextMate

This is a development Bundle for the TextMate editor, that allows quicker and easier development on Palm's webOS SDK.

Install

Simply double-click the tmbundle or move it to ~/Library/Application Support/TextMate/Bundles/. If you would like to use the Emulator-features, like sending test-calls, you must re-configure the Emulator to use ssh-public-key authentication. You can try using the feature provided by the bundle itself to configure this authentication method.
